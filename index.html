Teaching Archive

This website (and the associated GitHub repository) archives webpages for courses I’ve taught. It’s intended to be both a record and a resource: if you teach math, you’re welcome to steal anything from here that you like. Please also feel free to get in touch with me if you’d like to chat about something you find here. For brevity, I’ve mostly only listed the most recent iteration of a given course at a given institution, and only when there may be something “worth seeing” on that page. However, this website does also archive other pages not linked below, and you can see what else is around by browsing the associated GitHub repository.
